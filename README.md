# 🧠 Alzheimer-IoT Solution

An IoT-based healthcare monitoring system designed to improve the safety and well-being of Alzheimer's patients using ESP32.

## 📌 Overview

Alzheimer's disease affects memory, cognitive abilities, and daily activities, often requiring continuous supervision. This project provides a smart healthcare monitoring solution that helps caregivers monitor patient conditions and respond quickly during emergencies.

## 🚀 Features

- 🌡️ Real-Time Temperature Monitoring
- 🚨 Emergency SOS Alert System
- 💊 Medication Reminder
- 📍 GPS Location Display (Sample Coordinates)
- 📺 OLED Display Interface
- 🔊 Buzzer Alert Notification
- 💡 LED Status Indicator
- 📟 Serial Monitor Logging

## 🛠️ Hardware Components

- ESP32 Development Board
- DHT22 Temperature Sensor
- SSD1306 OLED Display
- Push Button
- Buzzer
- LED
- Breadboard and Jumper Wires

## 💻 Software Used

- Arduino IDE
- Wokwi Simulator
- GitHub

## ⚙️ Working

1. The DHT22 sensor continuously monitors temperature.
2. Temperature and location data are displayed on the OLED screen.
3. A single button press triggers an emergency alert.
4. A double button press activates the medication reminder.
5. Alert information is displayed on both the OLED display and Serial Monitor.
6. LED and buzzer provide visual and audible notifications.
